# ▶️ How to Run

### For HTML, CSS & JavaScript Projects:

1. Clone or download the repository

2. Navigate to the required project folder

3. Open:

```
index.html
```

in any browser.

---

### For React Projects:

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm start
```
